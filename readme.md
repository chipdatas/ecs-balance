# 手动安装

1. 下载 源码里的 ali 到服务器上，
2. 给权限 `chmod +x ali`
3. 执行 `./ali `，首次访问会要求提示输入阿里云的信息，输入后会在同名目录下生成一个 `config.json` 文件，保存了阿里云的信息
4. 之后再执行 `./ali auto` 会尝试将这个程序加入到每次登录的时候执行，这样每次登录都会自动获取流量信息；如果不做这一步，也可以手动执行 `./ali` 来获取流量信息
